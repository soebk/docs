## Getting Started

### Step 1: Connect Your Wallet

1. Visit [chat402.xyz](https://www.chat402.xyz)
2. Click **"Connect Wallet"** in the sidebar
3. Choose your wallet:
   - **MetaMask** (Ethereum/Base)
   - **Phantom** (Solana)
4. Approve the connection

**What happens**: Chat402 automatically creates two signless custodial wallets for you:
- One on Ethereum (Base network) for USDC
- One on Solana for USDC

These wallets let you chat without signing every transaction!

### Step 2: Fund Your Wallet

You'll see two deposit addresses on the **API Keys** page:

#### For Ethereum (Base):
1. Copy your Ethereum deposit address
2. Send USDC to this address **on Base network**
3. Minimum: $1 USDC
4. Confirmation time: ~1-2 minutes

**Where to get USDC on Base:**
- Bridge from Ethereum using [bridge.base.org](https://bridge.base.org)
- Buy directly on Coinbase
- Use [Uniswap on Base](https://app.uniswap.org)

#### For Solana:
1. Copy your Solana deposit address
2. Send USDC to this address on Solana mainnet
3. Minimum: $1 USDC
4. Confirmation time: ~30 seconds

**Where to get USDC on Solana:**
- Use [Jupiter Exchange](https://jup.ag)
- Buy on Phantom wallet
- Bridge from other chains

## Advanced Features

### Using Your Own Wallet (x402 Protocol)

For advanced users who want to pay directly from their wallet:

```javascript
// Sign payment proof
const message = `PromptPort Payment\nRequest ID: ${requestId}\nAmount: ${amount}\nTimestamp: ${timestamp}`;
const signature = await wallet.signMessage(message);

// Send with payment proof
await fetch('https://api.chat402.xyz/api/v1/prompt', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    model: 'gpt-4',
    prompt: 'Your question',
    payment: {
      from: walletAddress,
      amount: 1000, // USDC cents
      signature,
      timestamp,
      requestId
    }
  })
});
```
