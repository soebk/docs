## Chatting with AI Models

Select from 10+ premium AI models and start chatting instantly.

### Selecting a Model

Available models in the tabs:
- **GPT-3.5 Turbo** - Fast, general-purpose ($0.0005/1K tokens)
- **GPT-4 Turbo** - Complex reasoning ($0.010/1K tokens)
- **Claude 4.5 Sonnet** - Analysis, long-form ($0.003/1K tokens)
- **Gemini 2.5 Flash** - Cost-effective ($0.0001/1K tokens)
- **Grok** - xAI's latest ($0.005/1K tokens)
- **DeepSeek Chat** - Code generation ($0.0001/1K tokens)

### Sending Messages

1. Type your message in the input box
2. Press **Send** or hit **Enter**
3. See the AI's response with cost breakdown
4. Continue the conversation

### Cost Tracking

Each response shows:
- Token count (input/output)
- Total cost in USD
- Model used

Your total balance is displayed in the header.
