## Private Key Export

Your keys, your crypto. Export anytime.

### How to Export

1. Go to **API Keys** page
2. Click **"Export Private Keys"**
3. Sign message to prove ownership
4. Get private keys for both chains

### Importing to Wallets

**MetaMask** (for Ethereum/Base):
1. Click account icon → Import Account
2. Paste private key
3. Select Base network

**Phantom** (for Solana):
1. Settings → Add/Connect Wallet
2. Import Private Key
3. Paste your Solana key

<Warning>
Never share your private keys! Anyone with these keys has full control over your wallets and funds.
</Warning>
